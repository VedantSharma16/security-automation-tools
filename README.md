# security-automation-tools

Security tools and small pipelines, built up over time.

## Projects

- [`log-sentinel/`](log-sentinel/) — rule-based SSH auth log detection
  (brute force, compromised-credential, off-hours, privileged-login rules)
  with LLM-assisted, MITRE ATT&CK-grounded alert triage. Includes tests and
  a runnable CLI demo.
- [`port_scanner.py`](port_scanner.py) — basic TCP connect-scan learning
  exercise.
- [`system_logger.py`](system_logger.py) — basic process-listing /
  keyword-matching learning exercise.
